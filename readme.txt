Nel file "Elaborato" è presente la tesi di quinta superiore incentrata sul Machine Learning, con focus su Reti Neurali.
Nel file "slp.js" è presente la pate di addestramento con una funzione lineare f:R -> R
Nel file "linear_regresione.js" è implementato il forward pass di un slp, ovvero la fase di calcolo durante la quale l'input viene processato attraverso la rete per produrre l'output.
